# Прошлые события JetStyle

## Провальные кейсы (для диагностики)

### AI Meetup (октябрь 2025)
**Формат:** Классический митап, 3 доклада по 20 минут
**Ожидали:** 50 человек
**Пришло:** 12 человек

**Что пошло не так:**
- Анонс за 5 дней до события
- Скучное название "AI в дизайне: тренды"
- Доклады были про "что такое AI" — аудитория уже знала
- Нет интерактива — люди слушали и уходили
- Не собрали контакты участников

**Canvas этого события (заполни пробелы):**
```

Contributions: Доклады
Motivations: ???
Enablers: ???
Rules: ???
Tools: Zoom, Telegram-чат
```

### Летняя стажировка 2024
**Формат:** 3 месяца, 10 студентов, 5 менторов
**Результат:** 2 человека остались работать

**Что пошло не так:**
- Слишком долго (3 месяца — большой commitment для менторов)
- Менторы выгорели к середине
- Студенты не чувствовали прогресс
- Нет community после — люди ушли и забыли
- ROI для компании: 500+ часов менторства → 2 джуна

**Цитата ментора:**
> "Я потратил 100 часов, а толку — один джун который через полгода ушёл"

### Design Drinks (март 2025)
**Формат:** Неформальная встреча дизайнеров, без программы
**Ожидали:** Organic networking, 30 человек
**Пришло:** 8 человек (5 из них — сотрудники JetStyle)

**Что пошло не так:**
- Нет структуры = нет повода прийти
- "Просто пообщаться" — слабая мотивация выйти из дома
- Не было понятно кто придёт и зачем
- Нет Creative Constraint — всё размыто

## Успешные кейсы (для сравнения)

### VR Demo Day (регулярно)
**Формат:** 3 часа, демонстрация VR-проектов, свободный вход
**Результат:** 15-20 человек стабильно

**Почему работает:**
- Уникальный опыт (попробовать VR)
- Низкий порог входа (просто приходи)
- Конкретная ценность (потрогать технологию)
- Есть wow-фактор

**Проблема:** Не масштабируется — 3 шлема = 20 человек максимум

---

### Хакатон для стартапов (2023)
**Формат:** 48 часов, 8 команд, прототипы для реальных стартапов
**Результат:** 40 участников, 2 стартапа стали клиентами

**Почему работает:**
- Конкретный результат (работающий прототип)
- Высокая мотивация (для реального бизнеса)
- Интенсив = высокий commitment
- Creative Constraint: "48 часов, рабочий прототип"

**Проблема:** Очень трудоёмко — делали раз и больше не повторяли

---

## Паттерны из анализа

| Работает | Не работает |
|----------|-------------|
| Конкретный результат | "Просто пообщаться" |
| Уникальный опыт | Ещё один митап |
| Низкий порог входа | Долгий commitment |
| Чёткий Creative Constraint | Размытый формат |
| Практика и интерактив | Только лекции |
| Wow-фактор | Скучные темы |
