# Social Design Foundations

Интерактивный курс по основам Social Design для Claude Code.

## Быстрый старт

### 1. Склонируй репозиторий

```bash
git clone https://github.com/kulakov/social-design-foundations.git
cd social-design-foundations
```

### 2. Открой в Claude Code

```bash
claude
```

### 3. Запусти первый модуль

```
/sd-1-1
```

## Что это

Тренажёр из 3 модулей (~70 минут), который учит проектировать кооперативные форматы (митапы, хакатоны, сообщества) по методологии Social Design Cookbook.

## Модули

| # | Команда | Тема | Время |
|---|---------|------|-------|
| 1.1 | `/sd-1-1` | Social Design Canvas — 6 элементов формата | 20 мин |
| 1.2 | `/sd-1-2` | Creative Constraints — правила как драйвер участия | 25 мин |
| 1.3 | `/sd-1-3` | Мотивации — 5 типов мотивации участников | 25 мин |

## Как устроен курс

Каждый модуль — это интерактивный диалог с Claude:

1. **Провальный кейс** — разбираем реальную проблему
2. **Микро-теория** — короткие объяснения по ходу
3. **Практика** — классификация, диагностика, генерация
4. **Чеклист** — проверка что усвоил

Все упражнения используют синтетический контекст компании JetStyle (digital-агентство, Екатеринбург).

## После курса ты умеешь

- Заполнять Social Design Canvas для любого формата
- Диагностировать почему формат провалился
- Создавать Creative Constraints (простые правила → массовое участие)
- Работать с 5 типами мотивации участников
