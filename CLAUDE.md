# Social Design Foundations — Тренажёр

Интерактивный курс по основам Social Design на основе Social Design Cookbook.

## Структура курса

```
social-design-foundations/
├── .claude/
│   ├── SCRIPT_INSTRUCTIONS.md    # Правила для обучающих скриптов
│   └── commands/                  # Slash-команды запуска модулей
├── company-context/
│   └── jetstyle/                  # Синтетический контекст для практики
├── lesson-modules/
│   └── 1-foundations/             # 3 модуля основ
└── course-structure.json          # Конфигурация курса
```

## Как начать

Запусти первый модуль:
```
/sd-1-1
```

## Доступные модули

### Level 1: Foundations (~70 минут)

| Модуль | Команда | Тема |
|--------|---------|------|
| 1.1 | `/sd-1-1` | Social Design Canvas (6 элементов) |
| 1.2 | `/sd-1-2` | Creative Constraints (NaNoWriMo-паттерн) |
| 1.3 | `/sd-1-3` | Мотивации участников (5 типов) |

## Что изучаем

После курса студент умеет:

- **Canvas**: Заполнять Social Design Canvas, диагностировать провальные форматы
- **Constraints**: Создавать Creative Constraints, проверять по 3 критериям
- **Мотивации**: Классифицировать 5 типов, проектировать форматы для разных мотиваций

## Контекст для практики

Все упражнения используют синтетический контекст компании **JetStyle** — digital-агентства из Екатеринбурга:
- Провальные кейсы (AI Meetup, летняя стажировка)
- Цитаты участников для классификации
- Сырые заметки и ограничения для упражнений

## Методология

Каждый модуль следует паттерну:
1. **Провальный кейс** — начинаем с реальной проблемы
2. **Микро-теория** — объяснение только когда нужно
3. **Skill-building** — действия формируют навык (классификация, диагностика, генерация)
4. **Чеклист** — самопроверка

## Источник

Курс основан на книге **Social Design Cookbook** (Artem Zheleznyakov).
